# Personal Portfolio Website

A modern, clean personal portfolio website built with Next.js and React.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the website.

## Deploying to Vercel

The easiest way to deploy this Next.js app is to use the [Vercel Platform](https://vercel.com).

1. Push your code to a GitHub repository
2. Import your repository on Vercel
3. Vercel will automatically detect Next.js and deploy your site

Alternatively, you can use the Vercel CLI:

```bash
npm install -g vercel
vercel
```

## Adding Your Photo

Place your profile photo in the `public` folder and name it `profile.jpg` (or update the image path in the code).

## Customization

- Update projects in `app/page.js`
- Modify styles in `app/globals.css` and `app/page.module.css`
- Add more sections as needed
